* haolian9/infra.nvim


## usage

my personal config
```
do --blood moon
  local spell = cmds.Spell("BloodMoon", function(args) assert(require("bloodmoon")[args.op])() end)
  spell:add_arg("op", "string", false, "show", cmds.ArgComp.constant({ "enable", "disable", "show" }))
  cmds.cast(spell)
end
```
